# WartaCalc

Mieux que :

- Excel
- LibreOffice Calc
- La caculatrice Windows
- Les TI-82 (mais pas les 83, ça c'est vraiment trop surpuissant)
- Toutes les CASIO
- Ton petit frère en CP qui sait pas ses tables de multiplication

## Dépendances

- Cargo et Rust
- Bash

## Utilisation

```
$ wc '2 + 2'
4

$ wc ""); println!("vive la sécurité"'

vive la sécurité
```
